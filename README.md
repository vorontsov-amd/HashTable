# HashTable

This is program, witch counts number of word in the text file.

## Instalation
For UNIX system compilling with makefile.
For Windows compilling with compiling with command line.

## How to use this programm

When running through the terminal, you can pass the name of the text file in which the program will read the words as an argument. By default, the program searches for a file with the name in its own directory text.txt
